# Delishop

The Delishop app creates instances of Order, Sandwich, Drink, and Chips, and uses a Scanner to read user input. It displays the home screen options (New Order, Exit) and manages the process of adding sandwiches, drinks, and chips to the current Order.It orchestrates the user interaction, menu navigation, and creation of Order and item objects. The Delishop app also handles order cancellation, displays order details, and saves the receipt to a file.
